# nalanda-project

Bibliophiles social forum where the users are not key rather the discussions around ideas is what matters.

## Reviving Nalanda

The basic feature is to act as a repository of books read, want to read, wishlist, and other such stuff.
Would transform into a book-based social network later on.
Discussions on books as a whole, certain ideas within them or paragraphs or even sentences, anything can be a discussion starter.

## Why this project

To practice some form of an original CRUD full stack application.
Also, a good jump start podium to other design systems.
